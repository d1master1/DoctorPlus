# DoctorPlus — информационная система частной клиники

Это учебный проект, реализующий информационную систему для частной клиники.

## Основные возможности:
- Регистрация и авторизация
- Управление пациентами, врачами, услугами и записями
- REST API с документацией через Swagger
- Разграничение ролей: USER, EMPLOYEE, ADMIN
- Bootstrap интерфейс

## Технологии:
- Java 17
- Spring Boot
- Thymeleaf
- Spring Security
- Spring Data JPA
- MySQL
- Bootstrap 5
- SpringDoc OpenAPI
